¿Qué contiene un Repositorio?

Un repositorio es esencialmente el lugar donde se almacena:

  • Todo el código fuente y archivos de tu proyecto (archivos HTML, CSS, JavaScript, Python, imágenes, documentación, etc.).
  • El historial completo de revisiones de cada uno de esos archivos.
